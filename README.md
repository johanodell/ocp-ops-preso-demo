# ocp-ops-preso-demo
repo used for introduction to openshift ops

This repo can be used to do demos that goes along with the OpenShift_Introduction_preso.pdf in this repo. 

For the demo magic script demo.sh to work you will need
- kitty or similar terminal (that can show pictures) 
- lolcat
- viu
- highlight
- "Red Hat say" (https://github.com/RedHatNordicsSA/redhatsay.git) 
- be logged into an openshift cluster
